## Data Sources

This project uses publicly available datasets from Kaggle:

- **Olist E-Commerce Dataset**  
  https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce  
  Contains orders, customers, sellers, payments, products, and reviews.

- **Olist Marketing Funnel Dataset**  
  https://www.kaggle.com/datasets/olistbr/marketing-funnel-olist  
  Contains marketing qualified leads (MQLs) and closed deals representing
  seller acquisition via marketing channels.

The datasets originate from different operational systems and are treated
as separate sources during ingestion.
